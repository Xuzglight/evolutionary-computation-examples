# 进化计算学习笔记与代码实践

本项目是我在学习进化计算（Evolutionary Computation）过程中，从零开始手写的一些小例子。包含遗传算法、进化策略、遗传规划等经典算法的核心实现。
希望能帮助正在学习这些内容的同学更直观地理解算法的运行机制，或者单纯功利性的应对课程作业。

## 项目简介
进化计算是一类受生物进化启发的随机优化与搜索技术。虽然从原理上来说学习不同的算法较为简单，但是我们往往会在实践中遇到许多问题。
因此，我创建了这个项目，我认为该项目可以在以下两个方面帮助到初学者：
- **从零实现**：尽可能少依赖第三方库，使用 `numpy` 和 `matplotlib` 进行核心逻辑和可视化。
- **教学导向**：每个算法都配有详细的中文原理解释、注释，并拆解了初始化、评价、选择、交叉、变异等关键步骤。同时在项目中还附上了相关算法的论文供有兴趣者进行研读
